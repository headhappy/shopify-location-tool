Update Product Locations

Scan a product barcode, then update shop-floor LOCATION and stockroom LOC2.

Variant metafield: stock.location. Use for where staff find the item on the shop floor. For last item on display, prefer TDC1 (LOD) instead of only LOD where possible.
Product metafield: custom.location. Use for the bulk/secondary stock area closest to stock management.
Ready.
Simple rule:
LOCATION = where it is on the shop floor. LOC2 = where the backup/bulk stock is. If LOC2 is empty and the final piece is on display, try using a readable location like TDC1 (LOD). If you only know it is last-on-display but not where, use LOD as a temporary warning.
